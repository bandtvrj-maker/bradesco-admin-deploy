# Guia de Deploy do Sistema Bradesco no Railway

Este documento detalha os passos necessários para realizar o deploy do sistema Bradesco (Painel Admin e Cliente) em uma nova conta do Railway. Ele assume que você já possui uma conta no Railway e acesso ao terminal.

## 1. Pré-requisitos

Certifique-se de ter as seguintes ferramentas instaladas em seu ambiente local:

*   **Node.js:** Versão 18 ou superior.
*   **pnpm:** Gerenciador de pacotes (alternativa ao npm/yarn).
    ```bash
    npm install -g pnpm
    ```
*   **Railway CLI:** Ferramenta de linha de comando do Railway.
    ```bash
    curl -fsSL https://railway.app/install.sh | sh
    # Adicione o Railway CLI ao seu PATH (se necessário)
    export PATH="$PATH:$HOME/.railway/bin"
    ```

## 2. Estrutura do Projeto

O projeto é composto por duas partes principais:

*   `client/`: Contém o frontend (Painel Admin em React/Vite) e os scripts injetados no cliente (`cliente-bridge.js`).
*   `server/`: Contém o backend (servidor Express/Socket.IO/tRPC) que gerencia as sessões, a comunicação em tempo real e serve o HTML do cliente.
